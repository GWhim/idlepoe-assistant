# idlepoe Assistant

IdlePOE 测试服助手用户脚本。

## Greasy Fork 同步

Greasy Fork 源代码同步建议绑定仓库中的固定入口文件：

```text
idlepoe-assistant.user.js
```

GitHub Raw 地址：

```text
https://raw.githubusercontent.com/GWhim/idlepoe-assistant/master/idlepoe-assistant.user.js
```

每次原作者脚本更新后，先把本地附加功能迁移到最新版本，再重新生成 `idlepoe-assistant.user.js` 并推送即可。

## 当前附加功能

- 技能石手机端多选列表优化，支持槽位中文显示和长文本换行。
- 自定义打造在剥离/崇高后打印当前全部词缀。
- 特殊条件 `是否有工艺多大师`，判断工艺后缀 `大师之`。
